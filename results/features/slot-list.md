# Отчёт о реализации фичи: Backend для SCR-01 «Расписание»

## 1. Цель реализации

Реализовать backend-часть главного экрана мобильного приложения «Скалодром» — экрана расписания тренировок (SCR-01). Backend предоставляет REST API для получения списка доступных слотов тренировок с фильтрацией по датам и зонам, а также корректно обрабатывает ошибки согласно общим техническим требованиям.

---

## 2. Покрытые требования

### 2.1. Функциональные требования (FR)

| Требование | Описание |
|---|---|
| FR-01 | Отображение слотов на ближайшие 7 дней от текущей даты |
| FR-02 | Фильтр/календарь для выбора дат за пределами 7-дневного окна |
| FR-03 | Карточка слота: время начала, зона/формат, ФИО инструктора, количество свободных мест |
| FR-17 | Empty State при отсутствии слотов |

### 2.2. Пользовательские сценарии (UC)

| Сценарий | Описание |
|---|---|
| UC-01 | Просмотр расписания тренировок |
