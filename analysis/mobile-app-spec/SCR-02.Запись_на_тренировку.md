# SCR-02. Запись на тренировку (Выбор снаряжения)

## Техническая реализация

### 1. Входные данные
- Передаётся объект `Slot` с экрана SCR-01
- Отображается краткая информация:
    - Дата (формат: "dd.MM.yyyy")
    - Время (формат: "HH:MM")
    - Формат (из `zone.name`)

### 2. Выбор снаряжения

#### 2.1. Компонент переключения
- **Switch** или **Segmented Control**
- Опции: "Со своим снаряжением" / "Взять напрокат"

#### 2.2. Отображение стоимости
- При выборе "Взять напрокат":
    - Запрос к `GET /equipment`
    - Отображение стоимости аренды (из `rental_price`)
    - Если `available_count = 0` → сообщение "Снаряжения для проката нет"

#### 2.3. Состояния
- **Загрузка**: отображается индикатор
- **Успех**: активна кнопка "Подтвердить запись"
- **Ошибка**: сообщение о недоступности снаряжения

### 3. Подтверждение записи

**API Endpoint**: `POST /bookings`
**Тело запроса**:
```json
{
  "slot_id": 123,
  "equipment_option": "own" // или "rental"
}
