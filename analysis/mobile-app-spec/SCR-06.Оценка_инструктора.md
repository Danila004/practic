# SCR-06. Оценка инструктора

## Техническая реализация

### 1. Входные данные
- Передаётся `booking_id` из SCR-04
- Отображается `instructor.full_name`

### 2. Компонент оценки

#### 2.1. Звёздная шкала
- 5 звёзд (от 1 до 5)
- Визуальная обратная связь при наведении
- Выбор только одной оценки

#### 2.2. Текстовое поле
- Опционально
- Максимальная длина: 500 символов
- Placeholder: "Оставьте комментарий (опционально)"

### 3. Отправка оценки

**API Endpoint**: `POST /bookings/{booking_id}/rating`
**Тело запроса**:
```json
{
  "stars": 4,
  "comment": "Отличная тренировка!"
}
