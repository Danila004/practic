# SCR-04. Мои записи

## Контент экрана
* Список всех броней клиента (активные, прошедшие, отмененные).
* Статусы записей, включая «Отменена скалодромом» с указанием причины (FR-13, UC-05).
* Баннер-уведомление внутри экрана: «Одна из ваших тренировок была отменена...» (если push не был доставлен) (UC-05).

## Основные действия
* Выбор конкретной записи для просмотра деталей или отмены (UC-03).
* Переход к форме оценки инструктора (для прошедших тренировок) (UC-04).

## Второстепенные действия
* Обновление списка (pull-to-refresh) для получения актуальных статусов с бэкенда (UC-05).
