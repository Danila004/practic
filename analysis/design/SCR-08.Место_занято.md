# SCR-08. Место занято (Конкуренция за слот)

## Контент экрана
* Модальное окно с текстом: «К сожалению, все места заняты. Выберите другой слот» (FR-08, UC-02).
* Кнопка «Выбрать другой слот» / «Понятно».

## Основные действия
* Возврат на экран расписания (SCR-01) без перезагрузки списка для выбора другого времени (UC-02).

## Второстепенные действия
* Закрытие алерта (свайпом или крестиком).
