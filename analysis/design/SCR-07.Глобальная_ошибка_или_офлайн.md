# SCR-07. Глобальная ошибка / Оффлайн-режим

## Контент экрана
* Иконка и текст ошибки: «Не удалось загрузить данные. Проверьте подключение и попробуйте снова» (UC-01, NFR-02).
* Кнопка «Повторить попытку».
* Плашка с указанием времени последнего обновления: «Данные обновлены [время]» (при наличии кэша) (NFR-10).

## Основные действия
* Повторная попытка загрузки данных / запрос к API (UC-01).

## Второстепенные действия
* Просмотр закэшированных данных расписания или записей (при слабом соединении) (NFR-10).
