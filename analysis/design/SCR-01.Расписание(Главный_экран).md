# SCR-01. Расписание (Главный экран)

## Контент экрана
* Адрес скалодрома (FR-19).
* Список доступных слотов на ближайшие 7 дней от текущей даты (FR-01).
* Фильтр/календарь для выбора дат за пределами 7-дневного окна (FR-02).
* Карточка слота: время начала, зона/формат тренировки, ФИО инструктора, количество свободных мест (FR-03).
* Empty state: текст «Пока нет доступных тренировок» при отсутствии слотов (FR-17).

## Основные действия
* Выбор свободного слота и переход к экрану записи (UC-01, UC-02).
* Применение фильтра для просмотра расписания на более длительный срок (UC-01).

## Второстепенные действия
* Переход в раздел «Мои записи» (навигация).
* Повторная попытка загрузки данных (при ошибке API или таймауте) (UC-01).
