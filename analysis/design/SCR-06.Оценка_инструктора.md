# SCR-06. Оценка инструктора

## Контент экрана
* ФИО инструктора.
* Звёздная шкала оценки (от 1 до 5 звёзд).
* Текстовое поле для ввода комментария (опционально).
* Кнопка «Отправить оценку».
* Состояние: благодарность «Спасибо за вашу оценку!» после отправки (UC-04).

## Основные действия
* Выбор количества звёзд и отправка данных оценки на бэкенд (UC-04).

## Второстепенные действия
* Закрытие формы без отправки (сохранение черновика локально для повторной попытки) (UC-04).
